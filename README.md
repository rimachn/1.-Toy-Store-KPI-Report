# 1.-Toy-Store-KPI-Report
A simple interactive KPI report to track key metrics and explore high-level trends.

**Project Type:** Data Analysis & Visualization (Power BI)

**Description:**  
Maven toys have multiple stores located in Mexico. Provided data contains transactional record from January 2022 to September 2023 and as a data analyst I have to create a dashboard for leadership team to monitor key business metrics and high-level trends.

Dashboard has three KPI cards that shows revenue, profit and total number of orders for September 2023 and compares it with previous month. Line chart shows revenue generated for each month, we can further drill through and see revenue generated by week and day. Donut chat gives us revenue generated by each product category. By looking at the bar char we can find which product has been mostly ordered. we can also explore KPI, trends and product details for each location by selection locations from slicer.

**Tools Used:**  
Power BI Desktop, Power Query , DAX, Excel

**Key Features:**  
- Store-wise and product-wise sales analysis  
- Profit and cost trend analysis  
- KPIs for revenue, profit margins, and sales growth  
- Interactive filters and slicers for user-driven insights
